1, pom.xml-be tegyük be:

<dependency>
	<groupId>org.aspectj</groupId>
	<artifactId>aspectjweaver</artifactId>
	<version>1.7.3</version>
</dependency>

2, Hozzunk létre egy osztály lehetõleg ..Aspect.java néven
	Tegyük rá:
		@Aspect
		@Component (Nem szükséges)

3, Engedélyezzük az AspectJ-t
	Ezt többféle képpen is meg lehet tenni, attól függ hogy xml, vagy java osztály alapú a projekt.
		Java Class:
			AppConfig.class:
				Annotáljuk meg az osztályt:
				@Configuration
				@EnableAspectJAutoProxy
				public class AppConfig {}
		XML:
			pl.: bans.xml:
				Nézet fül, és pipáljuk be a Namespaces-ben az aop-t
					Valami ilyen fog bekerülni: xmlns:aop="https://www.springframework.org/schema/aop"
				Meg lehet(kell) adni hogy csináljon bean-t a komponens annotációval ellátott osztályból.
					<context:component-scan base-package="com.epam-package_neve"><context:component-scan/>
				Be kell kapcsolni az AspectJ-t:
					beans.xml	
						<aop:aspect-autoproxy/>
						
4, Gyártsuk le a Bean-t az adott Apsetbõl
	Lépjünk bele az AppConfig.java-ba és hozzuk léte a beant:
		@Bean(name = "loggerAspect")
			public LoggerAspect loggerAspect() {
			return new LoggerAspect();
		}
