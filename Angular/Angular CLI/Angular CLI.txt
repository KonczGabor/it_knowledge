Install on windows from command line:
	npm install -g @angular/cli
	
To create a new Angular project:
	ng new projec-name 
	
	Születni fog a dependenciákkal egy:
		egy package.json a 
		
To run the DEV build of the app (stand in the app's folder):
	ng serve
	
	Mostmár elérhetõ:
		http://localhost:4200/
		
	Terminálás:
		CTRL+C
		
	Hogy prodot buildeljünk(létre fog jönni egy dist mappa a project folderen belül) -Optimalizált csomagok,
	ezekre fog mutatni az index.html
		ng build --prod
	
Az app, amivel dolgozunk egy routing module által generálódik, amit lehet konfigurálni acontact-managerrel:
	ng new contact-manager --style=scss --routing
	
	
	"extends": "../..tsconfig.json"			//Két mappával följebb a tsconfig.json filet bõvíti
