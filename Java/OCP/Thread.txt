A Thread class:

-implementálja a Runnable interfészt.

-nem absztrakt.
-----------------------------


-Akkor indul új thread, ha egy alkalmas objektumon (Thread -re kasztolható) meghívjuk a start() -ot.

Így hohzahtunk létre Thread-et:

	new osztályAmiExtendeliAThreadet();
,vagy:
	new Thread(osztlyAmiImplementáljaARunnablet);
-----------------------------


 public void run() indul el a start() hatására.
