Wildcard:Megszorítás a generics típusát illetõen:


Unbounded wildcard
	?
List<?> l =new ArrayList<String>();



Wildcard with an upper bound
	? extends type
List<? extends Exception> l =new
ArrayList<RuntimeException>();

ArrayList<Number> list = new ArrayList<Integer>(); // DOES NOT COMPILE
Instead, we need to use a wildcard:
List<? extends Number> list = new ArrayList<Integer>();
Felsõ szint, az, meg ami alatta van.



Wildcard with a lower bound
	? super type
List<? super Exception> l =new
ArrayList<Object>();
Alsó határ, az, meg ami fölötte van
