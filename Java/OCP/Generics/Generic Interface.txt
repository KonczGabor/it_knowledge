public interface Shippable<T> {
	void ship(T t);
}

Egy osztály háromféleképpen implementálhatja:

1,
class ShippableRobotCrate implements Shippable<Robot> {
	public void ship(Robot t) { }
}

2,
class ShippableAbstractCrate<U> implements Shippable<U> {
	public void ship(U t) { }
}

3, Ilyenkor raw type jön létre, a fordító figyelmeztet, de attól még lefordul, ez a régi módszer.

class ShippableCrate implements Shippable {
	public void ship(Object t) { }
}

Amit nem lehet:

	A type erausre miatt nem lehet hívni konstruktort: new T(), mert runtime ban ez new Objectet eredményezne.

	Statikus típusok tömbjét létrehozni //?

	Meghívni az instanceof -ot.

	Primitív típust használni generikus paraméterként.

	Statikus változót adni generikus paraméterként.
