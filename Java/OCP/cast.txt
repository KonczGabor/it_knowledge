subclassból superclassba nem szükséges a kasztolás.

Visszafele, superclassból subclassba szükséges a cast.

A fordító az egymással nem rokonsági kapcsolatban álló osztályok kasztolására fordítási hibát dob. 
public class Fish {
	public static void main(String[] args) {
		Fish fish = new Fish();
		Bird bird = (Fish)bird; // DOES NOT COMPILE
	}
}


Interface-re castoálsok átmehetnek ezen a checken és csak runtime szállnak el.
